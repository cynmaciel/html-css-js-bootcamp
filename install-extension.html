How to install and Use VS Code's Live Server Extension

Install | Use

Installation Instructions

  1. In Visual Studio Code, click the click the Extentions icon in the left column toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server Extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In the VS Code's Explorer, right click (Windows) or Control+click (macOS) on an HTML file. From the content menu that appears, select Open with Live Server.
